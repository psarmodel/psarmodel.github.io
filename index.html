Probabilistic System for Association Rules

Probabilistic System for Association Rules (PSAR) model is a formal approach to capture association rules with probabilistic behavior.

To cite PSAR, please use the most recent submitted paper:
  • Abdelhakim Baouya Mining of Explainable Anomaly Patterns via Association Rules and Probabilistic Transitions System. In IEEE transaction in software Engennering,2023. [pdf] [bib]
